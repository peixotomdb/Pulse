## Como Usar

### Web
- Coloque os favicons na pasta raiz do seu site
- Adicione as meta tags no <head> do seu HTML
- Use o manifest.json para PWA

### iOS
- Importe os ícones no Xcode
- Configure no Info.plist

### Android
- Coloque os ícones nas pastas res/drawable correspondentes
- Configure no AndroidManifest.xml
